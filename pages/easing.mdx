# Easing

<span className="lead">
  A set of functions for adjusting the feel of tween
  animations.
</span>

<div>

[Tween animations](/api/animation/#tween) represent their progress as a value from `0` to `1`. If we pass this value to an easing function, we can adjust it to change the animation in different ways as it progresses.

For instance, with the `Easing.expoOut` easing, we can start the animation fast and finish it slow. This kind of easing is useful for replicating natural motion.

</div>

```jsx
import * as React from "react"
import { Frame, Easing } from "framer"

export function MyComponent() {
  return (
    <Frame
      animate={{ rotate: 360 }}
      transition={{ ease: Easing.steps(5) }}
    />
  )
}
```
